# GEA Copilot Assistant - 部署指南

## 概述

GEA Copilot Assistant 是一个智能企业助手系统，集成到 GEA EOR SaaS 平台中，为运营人员提供 AI 驱动的数据分析和决策支持。

## 核心功能

### 1. 智能悬浮球界面
- **拖拽式悬浮球**: 可自由定位的 AI 助手入口
- **快捷键支持**: Ctrl+Shift+C 快速打开/关闭
- **场景感知**: 根据当前页面自动调整功能
- **多层级面板**: 聊天、预测、快捷操作分层展示

### 2. 智能聊天系统
- **多模态输入**: 支持文本、图片、文件上传
- **上下文理解**: 基于用户角色和当前页面提供精准回答
- **权限控制**: 严格遵循 RBAC 权限模型
- **成本透明**: 显示每次 AI 调用的成本和处理时间

### 3. 预测与预警引擎
- **薪酬截止预警**: 自动检测薪酬处理截止日期风险
- **休假审批监控**: 识别休假申请积压情况
- **财务异常检测**: 发现逾期发票等财务问题
- **员工状态分析**: 监控试用期员工等关键指标

### 4. 一键快捷操作
- **角色定制**: 根据用户角色显示相关快捷操作
- **智能推荐**: 基于使用习惯推荐常用功能
- **热键支持**: 支持键盘快捷键快速执行
- **上下文感知**: 当前页面相关的快捷操作

## 部署步骤

### 1. 数据库迁移
```bash
# 生成并应用数据库迁移
pnpm db:push
```

### 2. AI 任务配置
在 AI 设置页面配置以下任务类型：
- `copilot_chat` - 基础聊天
- `copilot_data_analysis` - 数据分析
- `copilot_file_analysis` - 文件分析
- `copilot_report_generation` - 报告生成
- `copilot_insights_extraction` - 洞察提取
- `copilot_payroll_analysis` - 薪酬分析
- `copilot_leave_analysis` - 休假分析
- `copilot_financial_analysis` - 财务分析

### 3. 权限配置
确保用户具有以下权限：
- **管理员**: 所有功能
- **运营经理**: 薪酬、休假、员工数据
- **财务经理**: 发票、财务数据
- **客户经理**: 客户、合同数据

### 4. 缓存配置
系统使用 LRU 缓存优化性能：
- 用户配置: 5分钟 TTL
- 对话历史: 10分钟 TTL
- 智能预测: 2分钟 TTL（实时更新）
- AI 响应: 10分钟 TTL

## 使用指南

### 快速开始
1. **打开 Copilot**: 点击右下角悬浮球或使用快捷键 `Ctrl+Shift+C`
2. **开始对话**: 输入问题或上传文件
3. **查看预测**: 系统自动显示相关预警信息
4. **使用快捷操作**: 点击快捷按钮或使用热键

### 高级功能
- **文件分析**: 支持 PDF、图片、文档的智能分析
- **数据导出**: 一键导出当前页面数据
- **自定义配置**: 在设置中个性化 Copilot 行为
- **快捷键自定义**: 配置个人喜好的快捷键

### 最佳实践
- 使用具体的问题描述获得更准确的回答
- 定期查看智能预测，及时处理潜在问题
- 利用快捷操作提高工作效率
- 根据角色权限合理使用功能

## 故障排除

### 常见问题
1. **Copilot 无法打开**: 检查浏览器控制台错误
2. **AI 响应慢**: 查看网络连接和服务器状态
3. **权限错误**: 确认用户角色和权限配置
4. **文件上传失败**: 检查文件大小和格式限制

### 日志查看
- 浏览器控制台: F12 打开开发者工具
- 服务器日志: 查看应用日志文件
- AI 调用日志: 在 AI 设置页面查看
