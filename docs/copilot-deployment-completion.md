## 🚀 部署步骤

### 1. 环境准备
```bash
# 安装依赖
pnpm install

# 应用数据库迁移
pnpm db:push
```

### 2. 配置设置
- **AI任务配置**：在AI设置页面配置8种Copilot任务类型
- **权限配置**：确保用户具有相应的角色权限
- **缓存配置**：LRU缓存已自动配置，无需额外设置

### 3. 启动服务
```bash
# 开发环境
pnpm dev

# 生产环境
pnpm build
pnpm start
```

### 4. 功能验证
- 访问任意管理页面，右下角会出现Copilot悬浮球
- 使用快捷键 `Ctrl+Shift+C` 打开/关闭
- 测试聊天功能、文件上传、预测显示等核心功能
